1.Node的环境搭建

-----首先要下载 Node.js的安装包----- （分为多个平台，根据不同平台自行选择，下面的例子是windows环境下）


https://nodejs.org/en/download/


-----安装-----


傻瓜式安装，存储位置可以选择 ，安装完了之后这要说一下，windows的 node控制台用起来很不舒服。windows推荐使用
 msysgit 下载地址： https://git-for-windows.github.io
 同样是傻瓜式安装不需要更改什么，安装完之后
 菜单栏里可以看到Git Bash，输入node --version 可以看到输出一个版本号，OK成功。
